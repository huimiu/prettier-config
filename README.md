# Prettier Config

## Configuration
```json
{
  "tabWidth": 2,
  "arrowParens": "always",
  "bracketSpacing": true,
  "endOfLine": "lf",
  "printWidth": 100,
  "semi": true,
  "singleQuote": false
}
```

## How to use
Add this to `package.json`:

```json
{
  "prettier": "@huimiu/prettier-config"
}
```
